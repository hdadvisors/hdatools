hdatools Ramp Lab — continuous ramp candidates (Session 2 / item 2.2)

Every ramp's light end (sequential) / center (diverging) anchors to a shared brand-wide hint of cream (H~80, C~10) instead of pure white or a same-hue tint — a real hue shift, not just low chroma — so light classes stay visible against a white page/basemap; tune it per-card via the "cream" sliders. HFV's Leaf and Cerulean anchors are experimental secondary colors (not part of hdatools:::.brands) swapped in for a weaker original pick — see each card's nudge note for what they replaced. Map/bar/line preview data is a single fixed synthetic dataset shared across all candidates so only the palette varies. CVD simulation uses the Machado 2009 linear-RGB transform matrices (an approximation for live feedback — verify any picked candidate against real colorspace::simulate_cvd() before finalizing).

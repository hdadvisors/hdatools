# Flip default major gridlines from horizontal to vertical

Flip default major gridlines from horizontal to vertical

## Usage

``` r
flip_gridlines(
  color = "#cbcdcc",
  linewidth = 0.05,
  size = lifecycle::deprecated()
)
```

## Arguments

- color:

  Gridline color

- linewidth:

  Gridline width

- size:

  **\[deprecated\]** Use `linewidth` instead.
